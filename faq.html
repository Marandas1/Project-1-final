Q:

How do I install and uninstall PunkBuster?

A:

The PunkBuster client software is installed when you install your game. If you chose not to install PunkBuster at the time the game was installed or if you would like to reinstall PunkBuster, please download our PBSetup utility and follow the instructions.

Q:

How do I get PunkBuster for a game that does not currently have it?

A:

The PunkBuster anti-cheat system is directly integrated into a game's engine. For this reason, Even Balance, Inc. works directly with a game's publisher/developer to integrate PunkBuster. There is no way to run PunkBuster in a game without tightly integrating it into the engine.If the game you are playing does not currently have anti-cheat measures, ask the publisher why not?
