Setup

    PunkBuster is designed to keep itself up to date using our proprietary auto-update system. For cases where the auto-update fails, we have created a utility named PBSetup that can manually ute one or more installations of PunkBuster
    To use the PBsetup
  • Download the appropriate version of PBSetup below
  • Run PBSetup
  • Click "Add a Game"
  • Choose the game you wish to update and confirm the installation path
  • Click "Update PunkBuster" to update the installations
